# Projektplan: Aufgabenlöscher

## Ziel
Entwicklung einer 3-Schichten-Webanwendung zur Aufgabenverwaltung.

## Technologien
- Frontend: React + Vite
- Backend: Python FastAPI
- Datenbank: SQLite
- Kommunikation: REST API + JSON

## Anforderungen (Pflichtenheft)
- [ ] Aufgaben hinzufügen
- [ ] Aufgaben anzeigen
- [ ] Aufgaben als erledigt markieren
- [ ] Aufgaben löschen
- [ ] Datenpersistenz gewährleisten

## Projektphasen
1. Tag 1: Setup & Planung
2. Tag 2: Datenbank & Backend-Logik
3. Tag 3: REST API & Frontend-Init
4. Tag 4: UI & Integration
5. Tag 5: Dokumentation & Abgabe
